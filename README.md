# SharePoint Permission Levels
<ul>
  <li>Site Collection Administrator</li>
  <li>Flil Control - includes owner level permissions including deleting site</li>
  <li>Design - includes owner level permissions excluding deleting site</li>
  <li>Edit - add, edit and delete the data</li>
  <li>Contribute - add and edit data</li>
  <li>View - view and download the data</li>
  <li>Read - only view the data</li>
  <li>Limited Access - applicable when access is granted to a single line item or a single document or folder and noton the whole site</li>
</ul>
